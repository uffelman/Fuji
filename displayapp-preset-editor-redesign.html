Design notes

  • Sheet is 500pt wide, matching the parent settings window
  • Section labels use same 10.5px uppercase style as About tab metadata labels
  • Form cards use same controlBackgroundColor + 10px radius as General tab
  • Display cards group the header (name + icon + badge) and dropdown in one card
  • Text input uses rounded border matching macOS native fields
  • Action bar right-aligned with Cancel (ghost) + Save (primary accent)
  • Save button is disabled when no name or no modes are selected
  • Shortcut badge uses the same monospace pill style as the presets list
